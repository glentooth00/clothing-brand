About This Project

This project is a Laravel-based web application for a clothing brand. It features both a CMS for administrators and a frontend display page for customers. The CMS allows management of products, categories, sales, and other content, while the frontend dynamically showcases the brand’s collections.

Key Features
Admin / CMS

User authentication and role-based access.

Dashboard with quick stats and navigation.

Manage products, categories, and featured items.

Upload images and manage content dynamically.

Track transactions and sales.

Frontend / Display Page

Showcase products and collections dynamically.

Search and filter products.

Responsive and mobile-friendly layout.

Smooth navigation menus and dropdowns for categories.

Technologies Used

Laravel 10 – Backend framework

PHP 8+

TailwindCSS – Frontend styling

Bootstrap Icons – Icon library

MySQL / MariaDB – Database
